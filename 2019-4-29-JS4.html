1.背景介绍

用return语句来终止一个函数的执行，并且返回expression的值，如果expression被省略，或者在函数中没有return语句被执行，则把undefined赋值给调用当前函数的表达式

return语句从当前的函数退出，并给那个函数返回一个值。

2.知识剖析

语法及返回方式

1，返回函数结果，语法为return与表达式，语句结果函数的执行，返回调用的函数，而且把表达式的值作为函数结果返回出去。

返回控制无函数结果，语法为return。在大多数情况下，为事件处理函数返回false，可以防止默认的事件行为，例如，默认情况下点击一个超链接，、 页面会跳转到，该元素href属性指定的页面

在这个时候，return false就相当于终止符，而return true就相当于执行符

在onclick事件中，除了我们指定意以外，还要触发一个事件执行页面的跳转，所以如果你想取消对象的默认动作的话，就可以使用return false。

返回函数结果


                function test() {
                    var a=1;
                    var b=7;
                    return a;//return 中止了函数的运行。并且把表达式的值返回给函数。
                    console.log(b);//undefined
                }
                    console.log(test());//1

返回函数控制


                function test() {
                    var a=1;
                    var b=7;
                    return a+b;//return 中止了函数的运行。并且把表达式的值返回给函数。
                    // console.log(b);//undefined
                }
                function demo() {
                    console.log(test());//8
                    }
                demo();

通过这两个栗子，我们可以明显的看出，return可以阻止函数的继续运行下去，并且会给该函数的表达式的值返回给这个函数。

3.常见问题

如果return在for循环中，还会执行一次循环吗？

解决方案


                function test() {
                    var demo=[];
                    for (var i=0; i<10; i++){
                    demo.push(i);
                    return demo;
                    }
                }
                console.log(test());//[0];
                //很显然，for循环只是返回了return所返回的值，并没有进行下次循环。所以，return可以让for循环跳出循环。

5.扩展思考

还要什么办法可以跳出for循环？

除了return可以跳出for循环，还有，break，以及continue。

break语句会使运行的程序立刻退出包含在最内层的循环，或者退出一个switch语句，由于它是用来退出循环的，或者switch语句，所以当他出现在这些语句时，这种形式的break才是合法的。

如果一个循环终止的条件特别复杂的情况下，那么我们使用，break语句来实现某些条件比用循环表达式要容易得多。

continue语句，continue语句与break语句相似，但是他不是一个退出循环，而是开始循环的一次新迭代。

return语句用于指定函数返回的值，return语句只能出现在函数体内，出现在代码的其他任何地方都会造成语法错误。且，当执行return语句的时候，即使函数还有其他语句，函数执行也会停止。
