1.背景介绍

Vue组件是什么

组件是可复用的 Vue 实例, 如果网页中的某一个部分需要在多个场景中使用，那么我们可以将其抽出为一个组件进行复用。组件大大提高了代码的复用率。

2.知识剖析

组件的组织

通常一个应用会以一棵嵌套的组件树的形式来组织

注册组件

全局注册

          
            Vue.component('demo', {
              template: '<div><h1>demo</h1></div>'
            })

记住：全局注册的行为必须在根 Vue 实例 (通过 new Vue) 创建之前发生。

局部注册

            
              var demo2 = {
                template: '>div>demo2>/div>'
              }
              new Vue({
                el: '#app',
                components: {
                  demo2
                }
              })
